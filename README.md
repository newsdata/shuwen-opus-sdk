# SHWOpus

## 1 如何接入

### 1.1 在项目的 Podfile 中，添加如下依赖：

```ruby
target 'YourProject' do
  pod 'SHWOpus', '~> 0.3.0'
end
```

### 1.2 执行 `pod install`

## 2 Requirements

Deploy target : iOS 8.0.
